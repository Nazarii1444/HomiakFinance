# Project Charter — Homiak Finance

Застосунок для ведення особистих фінансів з підтримкою кількох валют.

---

## 1. Проблема

Люди, які хочуть стежити за своїми грошима, не мають зручного місця де можна бачити все разом — скільки витратив, скільки заробив, скільки є на рахунках у різних валютах. Зазвичай це або складні програми не для звичайної людини, або таблички в Excel де все треба робити вручну.

**Конкретні проблеми:**

- Немає одного місця щоб бачити всі рахунки і витрати, особливо якщо є гривні, долари і євро одночасно
- Курси валют треба дивитись вручну і самому рахувати
- Немає можливості поставити собі фінансову ціль і відстежувати як ти до неї рухаєшся
- Немає нагадувань коли, наприклад, витратив більше ніж зазвичай або курс різко змінився
- Немає зрозумілої картинки — куди взагалі йдуть гроші

---

## 2. Цілі, результати та критерії успіху

### Цілі

| # | Ціль | Навіщо |
|---|------|--------|
| Ц-1 | Зручний облік у кількох валютах | Щоб не рахувати вручну і бачити все в одному місці |
| Ц-2 | Автоматичне оновлення курсів валют | Курси підтягуються самі з НБУ, не треба нічого вводити |
| Ц-3 | Фінансові цілі | Можна поставити ціль, наприклад «назбирати на відпустку», і бачити прогрес |
| Ц-4 | Сповіщення | Листи на пошту коли щось важливе — велика витрата, різка зміна курсу |
| Ц-5 | Звіти та аналітика | Зрозуміти куди йдуть гроші і як змінюється баланс |

### Що має бути в результаті

- Вебсайт де можна зареєструватися і вести свої фінанси
- Можливість додавати рахунки в різних валютах і записувати витрати та доходи
- Курси валют оновлюються автоматично раз на годину
- Листи на пошту про важливі події
- Сторінка зі звітами — куди витрачав, скільки залишилось
- Панель для адміністратора щоб слідкувати за роботою застосунку

### Критерії успіху

| Що перевіряємо | Очікуваний результат |
|----------------|----------------------|
| Швидкість завантаження | Відповідь сервера до 300 мс |
| Доступність | Застосунок працює 99.9% часу на місяць |
| Оновлення курсів | Повний цикл займає менше 2 хвилин |
| Сповіщення | Один і той самий лист не приходить двічі |
| Безпека даних | Дані зашифровані, є захист паролем і двофакторна перевірка |
| Офлайн | Можна переглядати останні 30 днів навіть без інтернету |

## 3. Аудиторія, стейкхолдери та обмеження

### Для кого це

| Хто | Хто це | Що йому потрібно |
|-----|--------|-----------------|
| Звичайний користувач | Людина яка хоче стежити за своїми грошима | Зручний облік, звіти, цілі, сповіщення |
| Користувач з кількома валютами | Хто має гривні, долари, євро або подорожує | Автоматичний курс, зрозумілий баланс у кожній валюті |
| Адміністратор | Технічна людина яка стежить за роботою застосунку | Слідкувати за помилками, керувати налаштуваннями |

### Стейкхолдери

| Хто | Роль | Що очікує |
|-----|------|-----------|
| HomiakTeam! | Команда розробки | Зробити все як описано, вкластися в терміни |
| Користувачі | Ті хто користуються застосунком | Щоб все працювало, було зручно і безпечно |
| Адміністратори | Ті хто обслуговують застосунок | Зручні інструменти для моніторингу |
| НБУ (API) | Дає курси валют | Щоб ми коректно використовували їх API |
| Gmail | Відправляє листи користувачам | Щоб ми не перевищували ліміти розсилки |
| GDPR | Закон про захист даних | Щоб ми дотримувалися правил обробки особистих даних |

### Обмеження

| Що | Як це впливає |
|----|---------------|
| Стек технологій зафіксований | Frontend — React, Backend — FastAPI, база даних — PostgreSQL. Міняти не будемо |
| Курси залежать від НБУ | Якщо НБУ недоступний — використовуємо останній збережений курс |
| Ліміти на листи | Gmail має обмеження скільки листів можна відправити |
| Закон про дані (GDPR) | Користувач може попросити видалити або вивантажити свої дані |
| Тільки сучасні браузери | Старі браузери не підтримуємо |
| Оновлення бази без зупинки | Будь-які зміни в базі даних не повинні призводити до відключення застосунку |

## 4. Product Vision

### Коротко про продукт

**Homiak Finance** — це вебзастосунок для людей, які хочуть розуміти що відбувається з їхніми грошима. Особливо зручно якщо є кілька рахунків у різних валютах — застосунок сам підтягує курси, показує зрозумілі звіти і нагадує про важливі зміни.

На відміну від банківських додатків, тут видно все разом і з будь-якого банку. На відміну від Excel — нічого не треба рахувати вручну.

### Структура Vision

| | |
|--|--|
| **Для кого** | Для людей які хочуть стежити за особистими фінансами, особливо в кількох валютах |
| **Яка проблема** | Немає зручного місця де все видно разом і нічого не треба рахувати вручну |
| **Що це** | Вебзастосунок Homiak Finance для особистого фінансового обліку |
| **Що дає** | Облік у кількох валютах, автоматичні курси НБУ, фінансові цілі, звіти і сповіщення |
| **Чим відрізняється від інших** | Простіше ніж корпоративні програми, розумніше ніж таблички в Excel |
