# User Stories з Acceptance Criteria

| UC | User Story | Acceptance Criteria |
|----|------------|--------------------|
| UC1 | Я, як користувач, хочу зареєструвати акаунт, щоб почати користуватися системою управління фінансами. | Користувач вводить email і пароль; система перевіряє валідність email; система створює акаунт; надсилається лист підтвердження. |
| UC2 | Я, як користувач, хочу підтвердити свою електронну пошту, щоб активувати обліковий запис. | Користувач відкриває посилання з email; система підтверджує адресу; акаунт активується. |
| UC3 | Я, як користувач, хочу увійти в систему, щоб отримати доступ до своїх фінансів. | Користувач вводить email і пароль; система перевіряє облікові дані; створюється сесія. |
| UC4 | Я, як користувач, хочу вийти із системи, щоб завершити свою сесію. | Користувач натискає logout; система завершує сесію. |
| UC5 | Я, як користувач, хочу скинути пароль, щоб відновити доступ до акаунта. | Користувач вводить email; система надсилає лист відновлення; користувач встановлює новий пароль. |
| UC6 | Я, як користувач, хочу змінити пароль, щоб підвищити безпеку свого акаунта. | Користувач вводить старий і новий пароль; система перевіряє коректність; пароль змінюється. |
| UC7 | Я, як користувач, хочу редагувати свій профіль, щоб оновити особисті дані. | Користувач змінює ім’я, email або валюту; система зберігає зміни. |
| UC8 | Я, як користувач, хочу видалити акаунт, щоб припинити використання системи. | Користувач підтверджує видалення; система видаляє дані користувача. |
| UC9 | Я, як користувач, хочу додати дохід, щоб відобразити отримані кошти. | Користувач вводить суму і категорію; система створює транзакцію доходу. |
| UC10 | Я, як користувач, хочу додати витрату, щоб відобразити свої витрати. | Користувач вводить суму і категорію; система зберігає транзакцію витрати. |
| UC11 | Я, як користувач, хочу зробити переказ між рахунками, щоб перемістити кошти. | Користувач обирає рахунки; система створює транзакцію переказу. |
| UC12 | Я, як користувач, хочу редагувати транзакцію, щоб виправити помилки. | Користувач змінює дані транзакції; система зберігає зміни. |
| UC13 | Я, як користувач, хочу видалити транзакцію, щоб прибрати некоректні дані. | Користувач підтверджує видалення; система видаляє транзакцію. |
| UC14 | Я, як користувач, хочу переглядати історію транзакцій, щоб бачити свої фінансові операції. | Система показує список транзакцій; доступна пагінація або прокрутка. |
| UC15 | Я, як користувач, хочу фільтрувати транзакції за датою, щоб знайти операції за певний період. | Користувач задає діапазон дат; система показує відповідні транзакції. |
| UC16 | Я, як користувач, хочу фільтрувати транзакції за категорією, щоб аналізувати витрати. | Користувач обирає категорію; система показує відповідні транзакції. |
| UC17 | Я, як користувач, хочу фільтрувати транзакції за рахунком, щоб переглядати операції конкретного рахунку. | Користувач обирає рахунок; система фільтрує список. |
| UC18 | Я, як користувач, хочу шукати транзакції, щоб швидко знаходити потрібні операції. | Користувач вводить ключове слово; система показує результати пошуку. |
| UC19 | Я, як користувач, хочу сортувати транзакції за сумою, щоб бачити найбільші витрати. | Користувач обирає сортування; система впорядковує список. |
| UC20 | Я, як користувач, хочу сортувати транзакції за датою, щоб бачити останні операції. | Користувач обирає сортування; система впорядковує список. |
| UC21 | Я, як користувач, хочу створити фінансовий рахунок, щоб відстежувати кошти. | Користувач вводить назву і тип рахунку; система створює рахунок. |
| UC22 | Я, як користувач, хочу редагувати рахунок, щоб змінити його параметри. | Користувач змінює дані рахунку; система зберігає зміни. |
| UC23 | Я, як користувач, хочу видалити рахунок, щоб прибрати непотрібний рахунок. | Користувач підтверджує дію; система видаляє рахунок. |
| UC24 | Я, як користувач, хочу переглядати баланс рахунку, щоб знати скільки коштів доступно. | Система показує актуальний баланс. |
| UC25 | Я, як користувач, хочу бачити місячні витрати, щоб аналізувати свої фінанси. | Система показує агреговані витрати по місяцях. |
| UC26 | Я, як користувач, хочу бачити місячні доходи, щоб аналізувати надходження коштів. | Система показує доходи по місяцях. |
| UC27 | Я, як користувач, хочу бачити витрати по категоріях, щоб зрозуміти структуру витрат. | Система показує статистику по категоріях. |
| UC28 | Я, як користувач, хочу бачити фінансові тренди, щоб аналізувати зміни у фінансах. | Система будує графіки трендів. |
| UC29 | Я, як користувач, хочу згенерувати фінансовий звіт, щоб експортувати статистику. | Користувач задає параметри; система генерує звіт. |
| UC30 | Я, як користувач, хочу створити фінансову ціль, щоб накопичити кошти на певну мету. | Користувач вводить суму і назву; система створює ціль. |
| UC31 | Я, як користувач, хочу редагувати фінансову ціль, щоб змінити її параметри. | Користувач змінює суму або опис; система зберігає зміни. |
| UC32 | Я, як користувач, хочу видалити фінансову ціль, щоб прибрати неактуальну мету. | Користувач підтверджує; система видаляє ціль. |
| UC33 | Я, як користувач, хочу переглядати список цілей, щоб бачити всі свої плани накопичення. | Система показує список цілей. |
| UC34 | Я, як користувач, хочу бачити прогрес цілі, щоб відстежувати наближення до мети. | Система відображає відсоток виконання. |

| UC37 | Я, як користувач, хочу конвертувати валюту, щоб бачити еквівалент суми. | Користувач вводить суму; система показує конвертацію. |

| UC41 | Я, як користувач, хочу отримати попередження про велику транзакцію, щоб контролювати незвичні витрати. | Система надсилає повідомлення. |
| UC42 | Я, як користувач, хочу отримувати повідомлення про зміну курсу валют, щоб реагувати на зміни ринку. | Система надсилає сповіщення. |
